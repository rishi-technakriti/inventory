This application has been developed for LAXMI MARKETING with a view to generate Estimate and Invoice, preparing database for Suppliers, Customers, Materials and to have a Stock Status of Materials.

  • MATERIAL RECEIPT

    This section will be used for entry of incoming materials. The Suppliers name, if exists in the database, will be shown and the address, mobile no. will be updated from the database. If the name is not in database, it will be updated with the new entry. Same is the case with item also.

  • MATERIAL SOLD

    This section will be used for entry of sold materials. The Customers name, if exists in the database, will be shown and the address, mobile no. will be updated from the database. If the name is not in database, it will be updated with the new entry.

  • STATUS

    This section is for checking and updating the data which has been entered in previous sections.

    • SUPPLIER’S PROFILE

      This is for updating the data of suppliers i.e. their name, address and mobile no.

    • CUSTOMER’S PROFILE

      This is for updating the data of customers i.e. their name, address and mobile no.

    • MATERIAL DATA

      This is for updating the data of material names

    • RECEIVED MATERIAL DATA

      This is for updating the data of material that has been received

    • EDIT INVOICE

      This is editing the invoice that has been saved earlier

    • REPRINT INVOICE

      This is reprinting of invoice that has been saved earlier

    • STOCK STATUS

      This will show the stock status of all materials that has been received and sold.

    • INVOICES

      This will show the invoices that has been generated
